# InsightBoard API

Backend API for **InsightBoard** – a SaaS-style Admin Dashboard with secure authentication and role-based access control.

Built to demonstrate production-ready backend architecture using Node.js, Express, MongoDB, JWT authentication, and middleware-based authorization.

---

## 🚀 Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- JSON Web Tokens (JWT)
- bcrypt (Password Hashing)
- Cookie-based Authentication
- Role-Based Access Control (RBAC)

---

## 🔐 Core Features

- User Registration & Login
- Secure password hashing with bcrypt
- JWT authentication stored in httpOnly cookies
- Protected routes middleware
- Role-based authorization (Admin/User)
- User management (CRUD operations)
- Centralized database configuration
- Clean MVC-style folder structure
